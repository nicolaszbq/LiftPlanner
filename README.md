# 🏋️ LiftPlanner

O **LiftPlanner** é uma aplicação focada em planejamento de treinos de musculação, permitindo organizar exercícios, estruturar rotinas e acompanhar a evolução de forma simples e eficiente.

> Projeto desenvolvido com fins educacionais e como parte do meu portfólio.

---

## 🚀 Funcionalidades

* 📅 Criação de planos de treino personalizados
* 💪 Organização de exercícios por grupo muscular
* 🔁 Estruturação de rotinas (ex: ABC, Full Body, etc.)
* 📊 Acompanhamento de progresso (cargas, repetições, etc.)
* 🧠 Interface simples e intuitiva

## ⚙️ Como rodar o projeto

```bash
# Clone o repositório
git clone https://github.com/nicolaszbq/LiftPlanner.git

# Entre na pasta
cd LiftPlanner

# Instale as dependências
npm install

# Rode o projeto
npm run dev
```
