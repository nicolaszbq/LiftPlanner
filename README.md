# 🏋️ LiftPlanner

O **LiftPlanner** é uma aplicação focada em planejamento de treinos de musculação, permitindo organizar exercícios, estruturar rotinas e acompanhar a evolução de forma simples e eficiente.

> Projeto desenvolvido com fins educacionais e como parte do meu portfólio.
